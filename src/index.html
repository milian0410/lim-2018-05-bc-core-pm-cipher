SECRET CIPHER

  • SecretCipher: Es una aplicación que sirve para reemplazar el orden de las letras del mensaje y viceversa y para eso utiliza el DESPLAZAMIENTO.
  • Desplazamiento: Es el número que vas a colocar para reemplazar a la letra original. Puede ser cualquiera. El N° que escojas sólo tu y/o el que recibe el mensaje deben tenerlo.
